# CredVault Demo

Credential management example.

## Create Credentials

```bash
# Create API key credential
aio sec create my-api-key -t apiKey --value '{"apiKey": "sk_live_xxx"}'

# Create database credential
aio sec create my-db -t database \
  --value '{"host": "localhost", "port": 5432, "username": "admin", "password": "secret", "database": "mydb"}'
```

## Read Credentials

```bash
# List credentials
aio sec list

# Get credential details
aio sec get my-api-key -t apiKey

# Reveal credential value
aio sec reveal my-api-key -t apiKey
```

## Cleanup

```bash
aio sec delete my-api-key -t apiKey --force
aio sec delete my-db -t database --force
```
